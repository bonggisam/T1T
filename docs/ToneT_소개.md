> **T**eacher **one** **T**aesung, 태성의 선생님 캘린더

---

## 🎯 만든 배경

학교 현장에서 선생님들은 매일 다양한 일정을 확인해야 합니다.

- 📋 교무실 게시판, 메신저, 구글 캘린더, 종이 시간표...
- 🔄 일정이 여기저기 흩어져 있어 놓치는 경우가 많고
- 🕐 컴시간 시간표는 매번 사이트에 들어가서 확인해야 하고
- 📱 개인 일정과 학교 일정을 따로 관리해야 하는 번거로움

**"컴퓨터를 켜면 바로 보이는 캘린더가 있으면 좋겠다"** 는 생각에서 ToneT이 시작되었습니다.

바탕화면에 자연스럽게 녹아드는 투명 위젯으로, 별도의 앱을 열지 않아도 학교 일정과 내 시간표를 한눈에 확인할 수 있습니다.

## ✨ 주요 기능

### 📆 공유 캘린더
- 학교 전체 일정을 **실시간 공유**
- 월 / 주 / 일 보기 전환
- 일정 **드래그로 간편 이동**
- 클릭 한 번으로 일정 추가

### 🔒 개인 일정
- 나만 볼 수 있는 **개인 일정** 관리
- 공유 일정과 함께 캘린더에 표시

### 📚 시간표 자동 연동
- **컴시간(Comcigan)** 시간표 자동 불러오기
- 이번 주 시간표가 캘린더에 시간대별로 표시
- **5분마다 자동 갱신** — 시간표 변경도 바로 반영

### 🖥️ 바탕화면 위젯
- **반투명 디자인**으로 바탕화면에 자연스럽게 배치
- 마우스를 올리면 편집 버튼 표시
- `Ctrl+Shift+C`로 편집 모드 빠른 전환

### 🔔 알림
- 새 일정 등록 시 **실시간 알림**
- 읽지 않은 일정 표시 (N 뱃지)

### 🎫 TPass 연동
- 상단 **T** 버튼으로 TPass 페이지 바로 열기
- 앱 안에서 캘린더 ↔ TPass 자유 전환

### 🎨 맞춤 설정
- 라이트 / 다크 / 시스템 테마
- 일정 글꼴 크기 조절
- 일정별 색상 구분

## 🛠️ 기술 스택

| 구분 | 기술 |
|------|------|
| 프레임워크 | Electron + React + TypeScript |
| 데이터베이스 | Firebase Firestore (실시간 동기화) |
| 인증 | Firebase Authentication |
| 시간표 | comcigan-parser (컴시간 연동) |
| 빌드 | electron-builder (Windows) |
