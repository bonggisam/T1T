# ToneT 교사 사용 안내서

> **ToneT** — 학교 일정 공유 캘린더 위젯  
> 바탕화면에 고정되어 학교 일정을 한눈에 확인할 수 있습니다.

---

## 1. 프로그램 설치

1. 아래 주소에서 최신 버전을 다운로드합니다.  
   **https://github.com/bonggisam/ToneT/releases**

2. 설치 방법 (택 1):
   - **`ToneT Setup x.x.x.exe`** — 설치 프로그램 (자동 업데이트 지원)
   - **`ToneT x.x.x.exe`** — 포터블 버전 (설치 없이 바로 실행)

3. 실행하면 바탕화면 우측 상단에 캘린더 위젯이 나타납니다.

---

## 2. 회원가입

### 가입 절차

1. 프로그램 실행 → 로그인 화면 → **「회원가입」** 클릭
2. 다음 정보를 입력합니다:
   - **이름** — 본인 실명 (캘린더에 표시됨)
   - **이메일** — 로그인에 사용할 이메일
   - **비밀번호** — 아래 조건을 모두 충족:
     - 8자 이상
     - 영문 포함
     - 숫자 포함
     - 특수문자 포함 (예: `!@#$%`)
3. **「가입 요청」** 버튼 클릭
4. 입력한 이메일로 **인증 메일**이 발송됩니다 → 메일의 인증 링크 클릭
5. **관리자 승인**을 기다립니다 (승인 완료 시 로그인 가능)

### 로그인

- 승인된 이메일과 비밀번호로 로그인합니다.
- 로그인 후 학교 공유 캘린더가 바로 표시됩니다.

---

## 3. 화면 구성

### 위젯 모드 (기본)
바탕화면에 고정되어 캘린더만 표시됩니다.  
마우스를 올리면 상단에 버튼이 나타납니다:

| 버튼 | 기능 |
|------|------|
| 📚 | 시간표 표시/숨기기 |
| **T** | TPass 열기/닫기 |
| ✏️ 편집 | 편집 모드 전환 |

### 편집 모드
**「✏️ 편집」** 버튼 또는 **`Ctrl+Shift+C`** 로 전환합니다.

| 버튼 | 기능 |
|------|------|
| 👥 | 관리자 패널 (관리자 전용) |
| 🔔 | 알림 패널 |
| 📚 | 시간표 표시/숨기기 |
| **T** | TPass 열기/닫기 |
| ⚙️ | 설정 |
| 📌 | 위젯 모드 (바탕 고정) |
| ─ | 최소화 |
| ✕ | 종료 (트레이로 이동) |

### 캘린더 보기 방식
상단 헤더에서 **월 / 주 / 일** 보기를 전환할 수 있습니다.

---

## 4. 일정 관리

### 일정 보기
- 캘린더에 **제목**이 표시됩니다.
- 마우스를 일정 위에 올리면 **시간, 카테고리, 작성자, 상세 내용**이 툴팁으로 표시됩니다.
- 일정을 **클릭**하면 상세 정보를 볼 수 있습니다.

### 일정 추가
1. 캘린더의 빈 영역을 **클릭**합니다.
2. 팝업에서 선택합니다:
   - **📅 공유 일정 추가** — 모든 교사에게 보이는 일정
   - **🔒 개인 일정 추가** — 본인만 볼 수 있는 일정
3. 제목, 시간, 설명 등을 입력하고 저장합니다.

> 💡 **주/일 보기**에서 클릭하면 해당 **시간대**에 맞춰 자동 설정됩니다.

### 일정 이동 (드래그)
- 본인이 만든 일정은 **마우스로 드래그**하여 날짜/시간을 이동할 수 있습니다.
- 월 보기: 날짜 단위 이동
- 주/일 보기: 시간 단위 이동

## 5. 시간표 연동 (컴시간)

컴시간(Comcigan)에 등록된 학교의 시간표를 캘린더에 자동으로 표시할 수 있습니다.

### 설정 방법

1. **편집 모드** → **⚙️ 설정** 클릭
2. **시간표 (컴시간)** 섹션을 찾습니다.
3. 다음 순서로 설정합니다:

#### ① 학교 검색
- 학교 이름을 입력하고 **「검색」** 클릭
- 검색 결과에서 **본인 학교를 선택** (클릭하면 파란색으로 표시)

#### ② 교사 정보 입력
- **교사 이름**: 본인 이름 입력 (예: `김철수`)
  > ⚠️ 컴시간에 등록된 이름과 동일해야 합니다 (앞 2글자로 매칭)
- **최고 학년**: 학교 유형에 맞게 선택
  - 중학교/고등학교 → `3학년`
  - 초등학교 → `6학년`

#### ③ 연동 시작
- **「🏫 시간표 연동 시작」** 클릭
- 설정 완료 후 **"주 N교시"** 로 수업 수가 표시되면 성공입니다.

### 시간표 표시

- **월 보기**: 오늘이 속한 주의 월~금에 시간표가 표시됩니다.
- **주 보기**: 이번 주 각 요일의 해당 시간대에 시간표가 배치됩니다.
- 📚 버튼으로 시간표 표시/숨기기를 전환할 수 있습니다.
- **5분마다 자동 갱신**됩니다 (주중 07:30~17:00).

### 시간표 초기화
설정 > 시간표 섹션에서 **「초기화」** 버튼으로 연동을 해제할 수 있습니다.

## 6. TPass 사용

상단의 **T** 버튼을 클릭하면 TPass 페이지가 앱 내에서 열립니다.

- **📅 캘린더** 버튼으로 다시 캘린더로 돌아갑니다.
- **🔄** 버튼으로 페이지를 새로고침할 수 있습니다.

---

## 7. 기타 기능

### 테마 변경
⚙️ 설정에서 **라이트 / 다크 / 시스템** 테마를 선택할 수 있습니다.

### 글꼴 크기 조절
⚙️ 설정에서 일정 표시 글꼴 크기를 조절할 수 있습니다.

### 트레이 아이콘
- 프로그램을 닫아도(✕) **트레이에 상주**합니다.
- 트레이 아이콘을 클릭하면 다시 표시됩니다.
- 트레이 우클릭 → **「종료」** 로 완전 종료할 수 있습니다.

### 키보드 단축키

| 단축키 | 기능 |
|--------|------|
| `Ctrl+Shift+C` | 편집 모드 ↔ 위젯 모드 전환 |

## 8. 자주 묻는 질문 (FAQ)

**Q. 가입 후 로그인이 안 됩니다.**  
A. 이메일 인증과 관리자 승인이 모두 완료되어야 로그인할 수 있습니다. 관리자에게 승인을 요청해 주세요.

**Q. 시간표가 보이지 않습니다.**  
A. 다음을 확인해 주세요:
1. ⚙️ 설정 > 시간표에서 **"주 N교시"** 가 표시되는지 확인
2. 📚 버튼이 활성화(진하게) 되어 있는지 확인
3. 컴시간에 등록된 이름과 설정한 이름이 일치하는지 확인
4. **「🔄 새로고침」** 버튼 클릭

**Q. 다른 교사의 일정을 수정/삭제할 수 있나요?**  
A. 아니요, 본인이 작성한 일정만 수정/이동/삭제할 수 있습니다. 관리자는 모든 일정을 관리할 수 있습니다.

**Q. 개인 일정은 다른 교사에게 보이나요?**  
A. 아니요, 개인 일정은 **본인만** 볼 수 있습니다. 개인 일정은 기기에 저장됩니다.

**Q. 프로그램 업데이트는 어떻게 하나요?**  
A. 설치 버전(Setup)은 자동 업데이트를 지원합니다. 새 버전이 있으면 상단에 업데이트 알림이 표시됩니다.
